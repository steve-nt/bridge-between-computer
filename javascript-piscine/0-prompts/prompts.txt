---

Make sure to incorporate comments within the code to explain each part clearly. Avoid using overly technical jargon that might confuse the student. 

---

Make sure to add either sources or keywords to incentivize furhter search and study from the students

The assignment is the following: 

More Or Less
Instructions
Create 4 functions:

more that takes 1 argument and adds 1 to it.
less that takes 1 argument and subtracts 1 from it.
add that takes 2 arguments and adds them together.
sub that takes 2 arguments and subtracts the second argument from the first.
