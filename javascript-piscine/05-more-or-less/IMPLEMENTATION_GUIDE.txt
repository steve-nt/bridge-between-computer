1. WHAT IS A FUNCTION?
   â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
   A function is like a machine in a factory. You feed it some input (called
   arguments or parameters), it processes that input according to its
   instructions, and then it gives you an output (the return value).

   Think of it this way:
   - A vending machine is a function:
     INPUT: money + button press (your arguments)
     PROCESS: the machine counts money and picks a snack (the function body)
     OUTPUT: a snack falls out (the return value)

   In JavaScript, a basic function looks like this:
   
   function nameOfFunction(argument1, argument2) {
       // code goes here
       return result;
   }


2. PARAMETERS vs. ARGUMENTS
   â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
   These terms are sometimes used interchangeably, but here's the difference:
   
   â€¢ PARAMETERS: the names you define when creating the function
   â€¢ ARGUMENTS: the actual values you pass when calling the function
   
   Example:
   function add(a, b) {    // 'a' and 'b' are PARAMETERS
       return a + b;
   }
   
   add(5, 3);              // '5' and '3' are ARGUMENTS


3. THE RETURN STATEMENT
   â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
   The 'return' keyword is how you send a value back from your function.
   Without it, your function will return 'undefined' (basically nothing).
   
   Once a return statement is hit, the function stops executing immediately.
   
   Example:
   function double(number) {
       return number * 2;  // This value goes back to whoever called the function
   }
   
   let result = double(5);
   console.log(result);    // prints: 10


4. ARITHMETIC OPERATORS
   â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
   These are the basic math symbols you'll be using:
   
   +  Addition:      5 + 3 = 8
   -  Subtraction:   5 - 3 = 2
   *  Multiplication: 5 * 3 = 15
   /  Division:      6 / 2 = 3

================================================================================

// FUNCTION 1: more
// Purpose: Increases a number by 1
// Parameters: num - the number to increase
// Returns: the number increased by 1
function more(num) {
    return num + 1;
}


// FUNCTION 2: less
// Purpose: Decreases a number by 1
// Parameters: num - the number to decrease
// Returns: the number decreased by 1
function less(num) {
    return num - 1;
}


// FUNCTION 3: add
// Purpose: Adds two numbers together
// Parameters: num1 - the first number
//             num2 - the second number
// Returns: the sum of num1 and num2
function add(num1, num2) {
    return num1 + num2;
}


// FUNCTION 4: sub
// Purpose: Subtracts the second number from the first number
// Parameters: num1 - the number to subtract from (the starting amount)
//             num2 - the number to subtract (the amount being taken away)
// Returns: the difference (num1 minus num2)
function sub(num1, num2) {
    return num1 - num2;
}
