                          KEY CONCEPTS EXPLAINED

CONCEPT 1: WHAT ARE ARRAYS AND STRINGS?
----------------------------------------

ARRAYS: Think of an array like a list or a playlist. It holds multiple items in
        a specific order.
        
        Example:  const fruits = ['apple', 'banana', 'orange'];
        
        It's like having boxes in a line, and each box contains a value.

STRINGS: A string is a sequence of characters (letters, numbers, symbols).
         Think of it like a sentence or a word.
         
         Example:  const greeting = 'Hello';
         
         Strings are like words written letter by letter in order.

WHY TREAT THEM SIMILARLY?
Because both have a FIRST and a LAST element! Both can be accessed by position.

---

CONCEPT 2: INDEXING - THE "ADDRESS SYSTEM"
-------------------------------------------

Every element in an array or string has an INDEX (a position number).

IMPORTANT: JavaScript counts starting from 0, NOT 1!

Think of it like house addresses on a street:
  - House Address 0: First house
  - House Address 1: Second house
  - House Address 2: Third house
  ... and so on

EXAMPLE - Array:
  const colors = ['red', 'green', 'blue', 'yellow'];
  
  Index:  0      1        2       3
  Value:  red    green    blue    yellow
  
  colors[0] = 'red'      (first element)
  colors[2] = 'blue'     (third element)
  colors[3] = 'yellow'   (last element)

EXAMPLE - String:
  const word = 'Pizza';
  
  Index:  0    1    2    3    4
  Value:  P    i    z    z    a
  
  word[0] = 'P'   (first character)
  word[3] = 'z'   (fourth character)
  word[4] = 'a'   (last character)

---

CONCEPT 3: THE .length PROPERTY
--------------------------------

Both arrays and strings have a LENGTH property that tells you how many items
(or characters) they contain.

EXAMPLE - Array:
  const colors = ['red', 'green', 'blue', 'yellow'];
  colors.length = 4   (there are 4 items)

EXAMPLE - String:
  const word = 'Pizza';
  word.length = 5     (there are 5 characters)

HOW TO GET THE LAST ELEMENT:
Since indexing starts at 0, the last element is always at index (length - 1).

  colors[colors.length - 1] = colors[3] = 'yellow'
  word[word.length - 1] = word[4] = 'a'

Think of it this way:
  If you have 5 items, they're at positions 0, 1, 2, 3, 4.
  The last one is at position (5 - 1) = 4. Perfect!

---

CONCEPT 4: RETURNING ARRAYS
----------------------------

Sometimes you need to return multiple values. The best way is to return a NEW
array containing those values.

Example:
  function getTwoNumbers() {
    return [5, 10];  // Returns a NEW array with two numbers
  }

You can then use the returned array:
  const result = getTwoNumbers();
  console.log(result[0]); // 5
  console.log(result[1]); // 10

STEP 1: BUILD THE first() FUNCTION
-----------------------------------

What does it need to do?
  - Accept either an array or a string
  - Return the item at index 0

Your approach:
  1. Create a function named 'first'
  2. Add a parameter to hold the input (let's call it 'collection')
  3. Return collection[0]

That's it! Because both arrays and strings can be accessed by index the same way.

---

STEP 2: BUILD THE last() FUNCTION
----------------------------------

What does it need to do?
  - Accept either an array or a string
  - Return the item at the LAST position

Your approach:
  1. Create a function named 'last'
  2. Add a parameter to hold the input
  3. Calculate the last index: collection.length - 1
  4. Return collection[collection.length - 1]

This works for both arrays and strings because both have a .length property!

---

STEP 3: BUILD THE kiss() FUNCTION
----------------------------------

What does it need to do?
  - Accept either an array or a string
  - Return a NEW array containing [last, first] in that order
  - Yes, it's "LAST FIRST" not "first LAST"!

Your approach:
  1. Create a function named 'kiss'
  2. Add a parameter to hold the input
  3. Use your last() and first() functions to get the last and first items
     (or write the logic directly)
  4. Return a new array: [last, first]

================================================================================
                        COMPLETE JAVASCRIPT CODE
================================================================================

// FUNCTION 1: first()
// Gets the first element from an array or first character from a string
function first(collection) {
  // Return the element at index 0 (the first position)
  // This works for both arrays and strings!
  return collection[0];
}

// FUNCTION 2: last()
// Gets the last element from an array or last character from a string
function last(collection) {
  // To get the last item, we use: array.length - 1
  // Why? Because indexing starts at 0, so the last index is always one less
  // than the total length.
  // 
  // Example: array of 4 items has indices 0, 1, 2, 3
  // So last index = 4 - 1 = 3
  return collection[collection.length - 1];
}

// FUNCTION 3: kiss()
// Returns a NEW array containing the last element and first element
// in that order: [last, first]
function kiss(collection) {
  // We create a new array with two elements:
  // - First element: the LAST item from the collection (using last function)
  // - Second element: the FIRST item from the collection (using first function)
  return [last(collection), first(collection)];
}
