SECTION 2: Device Selection [NECESSARY ‚úì]
================================================================================

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

NECESSITY: ‚úÖ NECESSARY (code will fail without this)

WHY IT'S ESSENTIAL:
  ‚úì Tell PyTorch WHERE to run computations
  ‚úì GPU (CUDA) is ~100x faster than CPU for neural networks
  ‚úì Without specifying device, code may fail when moving tensors

WHAT HAPPENS:
  
  With device specification:
    model = model.to(device)           ‚Üê Moves model to GPU/CPU
    images = images.to(device)         ‚Üê Moves data to GPU/CPU
    Result: Everything runs on same device ‚úì
  
  Without device specification:
    Error: Tensor device mismatch
    "Expected all tensors to be on the same device"

HOW IT WORKS:
  
  if torch.cuda.is_available():
    ‚Üí Uses GPU (CUDA) - Fast! ‚ö°
    ‚Üí device = "cuda"
    ‚Üí Forward pass: ~0.1 seconds per image
  else:
    ‚Üí Falls back to CPU - Slow but works üê¢
    ‚Üí device = "cpu"
    ‚Üí Forward pass: ~1-5 seconds per image
  
  In your case:
    You have GPU available (cu128 = CUDA 12.8)
    ‚Üí device will be "cuda"
    ‚Üí 10 images will take ~10 seconds (GPU)
    ‚Üí Without GPU would take ~50-100 seconds (CPU)
