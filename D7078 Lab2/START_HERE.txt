WHAT IS THIS LAB ABOUT?
================================================================================

CLOUD SECURITY:
   You will learn to secure cloud resources using:
   - Security Groups (firewall rules)

   - Analyzing latency and optimization

   After this lab you can:
   - Create secure cloud infrastructure
   - Write cloud applications in Java
   - Optimize cloud services for performance
   - Document cloud solutions professionally
