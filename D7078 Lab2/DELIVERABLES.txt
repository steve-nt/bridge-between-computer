QUICK START
===========

1. CONFIGURE AWS CLI
   aws configure
   aws sts get-caller-identity

2. GET YOUR IP
   curl https://checkip.amazonaws.com

3. RUN SETUP SCRIPT (Task 2.1)
   ./task-2-1-setup.sh YOUR_IP/32

4. COMPILE JAVA PROGRAMS (Task 2.2)
   javac -cp ".:aws-sdk/*" S3BucketOperations.java
   javac -cp ".:aws-sdk/*" S3LatencyMeasurement.java

5. RUN PROGRAMS
   java -cp ".:aws-sdk/*" S3BucketOperations
   java -cp ".:aws-sdk/*" S3LatencyMeasurement

6. REVIEW DOCUMENTATION
   - Implementation: Read lab-guide.md
   - Report Writing: Read lab-report-guide.md

7. CLEANUP (when done)
   ./task-2-1-cleanup.sh
