================================================================================
HOW TO INSTALL PYTHON ON WINDOWS
================================================================================

OPTION 1: DOWNLOAD FROM python.org (RECOMMENDED)
================================================================================

Step 1: Go to https://www.python.org/downloads/
Step 2: Click "Download Python 3.x.x" (latest version, e.g., 3.11 or 3.12)
Step 3: Run the installer

IMPORTANT: Check the box "Add Python to PATH" during installation!

Step 4: Click "Install Now"
Step 5: Wait for installation to complete
Step 6: Click "Close"

VERIFICATION:
Open PowerShell and type:
  python --version

Should show: Python 3.x.x (if installed correctly)

================================================================================

OPTION 2: INSTALL VIA WINDOWS PACKAGE MANAGER (winget)
================================================================================

If you have Windows 10/11 with winget installed:

Open PowerShell and run:
  winget install Python.Python.3.12

This automatically adds Python to PATH.

VERIFICATION:
  python --version

================================================================================

OPTION 3: INSTALL VIA CHOCOLATEY
================================================================================

If you have Chocolatey installed:

Open PowerShell as Administrator and run:
  choco install python

VERIFICATION:
  python --version

================================================================================

AFTER INSTALLATION: INSTALL REQUIRED PACKAGES
================================================================================

You need to install aiohttp (required for the load generator):

Open PowerShell and run:
  pip install aiohttp

Or if that doesn't work, try:
  python -m pip install aiohttp

VERIFICATION:
  python -c "import aiohttp; print('aiohttp OK')"

Should show: aiohttp OK

  ‚Üí Try: python -m pip install aiohttp

================================================================================

MANUAL PATH SETUP (If needed)
================================================================================

If Python is installed but not in PATH:

Step 1: Find Python installation folder
  Usually: C:\Users\YourUsername\AppData\Local\Programs\Python\Python311

Step 2: Add to PATH
  1. Press Windows + X
  2. Select "System"
  3. Click "Advanced system settings"
  4. Click "Environment Variables"
  5. Under "User variables", click "Path"
  6. Click "Edit"
  7. Click "New"
  8. Paste your Python path (e.g., C:\Users\YourUsername\AppData\Local\Programs\Python\Python311)
  9. Also add the Scripts folder: C:\Users\YourUsername\AppData\Local\Programs\Python\Python311\Scripts
  10. Click OK
  11. Restart PowerShell

Test again:
  python --version
